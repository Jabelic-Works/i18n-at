# Installation

## Prerequisites

- Node.js 16.8 or later
- Next.js 13.0 or later (with App Router)
- TypeScript 4.7 or later

## Package Manager

i18n-at can be installed with your favorite package manager:

### npm

```bash
npm install i18n-at
```

### yarn

```bash
yarn add i18n-at
```

### pnpm

```bash
pnpm add i18n-at
```
